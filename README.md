#  Twitter Sentiment Analyzer

A Machine Learning based web app that analyzes sentiment of tweets using FastAPI and Scikit-Learn.

##  Features
- Positive / Negative sentiment detection
- Confidence score
- Beautiful UI
- FastAPI backend

##  Tech Stack
- Python
- FastAPI
- Scikit-Learn
- HTML, CSS, JavaScript
